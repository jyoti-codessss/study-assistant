# Study Assistant 📚

An AI-powered Study Assistant built with Google ADK, deployed on Cloud Run.

## Features
- 🔍 Research any topic using Wikipedia
- 📝 Save study notes using MCP Filesystem tool
- 🌐 Fetch web content using MCP Fetch tool
- 🗄️ Store sessions in Firestore database
- 📅 Generate 7-day study schedules

## Tech Stack
- Google ADK (Agent Development Kit)
- Gemini 2.0 Flash
- Google Cloud Run
- Google Firestore
- MCP Tools (Filesystem + Fetch)
- Wikipedia API

## Deployment
Deployed on Google Cloud Run:
https://studyy-assistant-90810354691.asia-south1.run.app

## Usage
Ask the agent:
- "I want to study Artificial Intelligence"
- "Create notes for French Revolution"
- "Help me study Python programming"
